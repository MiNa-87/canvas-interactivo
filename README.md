# Canvas Interactivo · Té en Hebras

Herramienta web interactiva de **Modelo Canvas** para estudiantes del programa **Aprende a Emprender** de [Junior Achievement Argentina](https://www.jaargentina.org.ar/), aplicada a un emprendimiento real de bolsitas de té en hebras.

## ¿Qué es esta herramienta?

Una aplicación web educativa que permite a estudiantes secundarios completar su **Modelo Canvas** de forma guiada e interactiva, y luego simular los números básicos de su emprendimiento.

### Funcionalidades

- **9 bloques del Modelo Canvas** con preguntas guía adaptadas al emprendimiento de té en hebras
- **Guardado automático** en el navegador (localStorage)
- **Exportar** el Canvas completo como archivo de texto
- **Simulador financiero** con cálculo automático de:
  - Costos variables y fijos
  - Costo unitario real
  - Precio sugerido según margen de ganancia
  - Punto de equilibrio
  - Proyección realista de ventas del primer mes
- Diseño **responsive** (funciona en celular y computadora)
- Estética visual basada en la plantilla de Junior Achievement

## Cómo usarlo

Abrí `index.html` en cualquier navegador moderno. No requiere servidor ni instalación.

Para publicarlo online, podés usar [Vercel](https://vercel.com) o [GitHub Pages](https://pages.github.com) apuntando a `index.html` como raíz.
